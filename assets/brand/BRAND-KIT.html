Clear space + minimum size

Reserve a margin around the logo equal to the height of one wave glyph (~20% of the mark height). No element — text, photo, badge — encroaches inside that margin.

LockupMinimum width
Mark only16 px (favicon — anything smaller becomes a single dot)
Horizontal lockup120 px (~1 inch in print)
Stacked lockup80 px square

Don'ts

  • Don't stretch / squish the logo. Always uniform scaling.
  • Don't rotate or skew.
  • Don't swap the colors (no navy waves on teal box, etc).
  • Don't put the full-color mark on a high-contrast colored background — use the mono variant if the bg is neither white, cream, nor navy.
  • Don't use the wordmark in a different typeface ever. Always Fraunces 600.
  • Don't apply effects (drop shadows, gradients, glows) outside what's already in the SVG file.

5 · Voice + messaging rules

Vocabulary discipline is part of the brand. These rules are enforced in code (memory entries) and on every UI string we ship.

Forbidden words / phrases

"phone + email" → "owner data — name, phone, and/or email"
"free" (anywhere user-visible) → "no charge", "included", "covered"
"tenant" → "user", "account"
"Miami" (in marketing — except legal entity name) → "Florida"
"Resolve" (the verb) → "Enrich"
"propagated" → "same-owner match"
Any 3rd-party software brand without legal review → "our skip-trace vendor", "our payment processor"

The job-to-be-done framing

"You can't contact a prospect without their name, phone, and/or email. LeadCove solves this — quickly and simply."

This is the single sentence the brand is built around. Every marketing message either restates it, reinforces it, or follows from it.

Hard rules with money

  • "No data, no charge" — billing rule + marketing promise in lockstep. Misses don't bill.
  • Same product on every plan — only the included monthly credits change. No per-tier feature gating.
  • Customers bring their own list — we don't promise to source data.
  • Top-up packs are optional — never marketed prominently next to the monthly plans.
