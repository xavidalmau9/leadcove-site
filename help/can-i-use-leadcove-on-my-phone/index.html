Using the Dashboard

Can I use LeadCove on my phone?

Last updated 2026-05-06

Yes — the dashboard is mobile-responsive. Most operators prefer desktop for bulk actions, but call-cards, mark-as-contacted, and quick lookups all work fine on iPhone / Android browsers. There is no native mobile app yet.
